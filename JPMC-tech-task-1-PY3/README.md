# Task 1: Stock Price Data Feed Interface

## Overview
Real-time stock data feed server and client implementation in Python.

## Files
- server3.py - WebSocket server broadcasting stock data
- client3.py - Client receiving and displaying data
- client_test.py - Unit tests for client
- get-pip.py - Pip installation script
- test.csv - Sample test data
- solution.patch - Solution patch file
- *.pdf - Task documentation

## How to Run
# Start server
python server3.py

# In another terminal, run client
python client3.py

## Dependencies
Install requirements:
pip install -r ../requirements.txt
