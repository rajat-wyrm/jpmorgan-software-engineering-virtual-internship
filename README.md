# JPMorgan Chase Software Engineering Virtual Experience

This repository contains my implementation of tasks from the JPMorgan Chase Software Engineering Virtual Experience Program hosted on Forage.

## Overview

The project simulates a financial data feed system used by traders to analyze stock price discrepancies.

It processes market data and visualizes insights for trading analysis.

## Tasks Completed

Task 1 — Interface with Stock Price Data Feed  
Implemented a Python client that retrieves stock data and calculates price ratios.

Task 2 — Data Visualization with Perspective  
Integrated data visualization using a React + TypeScript component.

Task 3 — Trading Dashboard Visualization  
Implemented chart components to display financial insights.
