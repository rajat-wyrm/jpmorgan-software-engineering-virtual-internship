# JPMorgan Chase Software Engineering Virtual Experience

This repository contains my implementation of tasks completed during the JPMorgan Chase Software Engineering Virtual Experience Program on Forage.

## Project Overview

In this project, I worked with simulated financial market data to build tools that help traders visualize and analyze stock price discrepancies.

The system processes real-time stock data feeds and presents them through an interactive visualization interface.

## Key Features

- Stock price data feed integration
- Real-time data processing
- Financial data analysis
- Data visualization for trading insights

## Tech Stack

Python  
TypeScript  
React  
Git  

## Project Structure

task1-stock-data-feed/ → Data feed integration  
task2-perspective-integration/ → Data visualization setup  
task3-data-visualization/ → Trading dashboard charts
