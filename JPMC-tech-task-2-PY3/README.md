# Task 2: Perspective Integration

## Overview
TypeScript/Node.js server with Perspective visualization for real-time stock data.

## Structure
- src/ - TypeScript source files
- public/ - Frontend HTML/CSS/JS files
- datafeed/ - Data feed modules
- *.patch - Solution patch files
- *.pdf - Task documentation

## How to Run
# Install dependencies
npm install

# Start server
npm start

# Development mode with auto-reload
npm run dev

## Access
Open browser to http://localhost:8080
